# Trivial 🧩

A trivia quiz game for Android built with Kotlin and Jetpack Compose.
Choose your category and difficulty, answer 10 questions before the timer runs out, and share your score with friends.

## Download

[⬇️ Download APK](https://github.com/noagomezgarcia/Trivial/releases/latest)

## Features

- 6 question categories: History, Science, Sports, Geography, Entertainment and Art
- 3 difficulty levels: Easy, Medium and Hard
- 10-second countdown timer per question with animated progress bar
- Visual feedback on correct and incorrect answers
- Share your final score with any app via Android Intent
- Animated logo on the menu screen using Lottie

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose + Material 3
- **Architecture:** MVVM
- **State management:** StateFlow + collectAsStateWithLifecycle
- **Navigation:** Navigation3 with type-safe routes
- **Animation:** Lottie

## Getting Started

1. Clone the repository
```bash
   git clone https://github.com/noagomezgarcia/Trivial-Application.git
```
2. Open the project in Android Studio
3. Run on an emulator or physical device (Android 8.0+)
